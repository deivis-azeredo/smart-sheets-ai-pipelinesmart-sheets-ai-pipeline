# 📊 Smart Sheets AI Pipeline

> An automated, Python-driven solution for spreadsheet text validation and correction, built through advanced AI-Assisted Coding.

![Project Demo](automation_shhetsgif.gif)

## 🎯 The Problem
Manual review of large spreadsheets is time-consuming, repetitive, and prone to human error. Businesses need a way to process, validate, and correct text data instantly without relying on massive manual labor.

## 💡 The Solution
I engineered an automation pipeline where **Python acts as the execution engine**. By leveraging advanced Prompt Engineering with Large Language Models (Claude/Gemini), I developed the logic and generated the core Python scripts that handle the heavy lifting.

This tool automatically opens the spreadsheet, reads the target data, applies the necessary corrections, and saves the sanitized version in seconds.

## 🛠️ Tech Stack & Methodology
- **Execution Engine:** Python 
- **Development Methodology:** AI-Assisted Coding (Vibecoding) & Prompt Engineering
- **AI Models Used for Logic Building:** Claude 3.5 Sonnet / Google Gemini
- **Data Handling:** Spreadsheet automation libraries

## 🚀 Key Features
- **Zero Manual Entry:** Eliminates the need for line-by-line human review.
- **High Speed:** Processes large volumes of text rows instantly.
- **Error Reduction:** standardizes formatting and corrects data automatically.
